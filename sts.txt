자 내가 윈도우 bat를 만들고 싶어.
업무를 설명할께.

1. 시작

아래 정보를 일단 수집해야 해.
2. 현재 실행중인 폴더명이 [프로젝트명]이야
3. 현재 실행중인 폴더 아래 > src / main / java / 폴더명1 / 폴더명2 / 폴더명3 이 존재해
4. 폴더명1.폴더명2 가 [패키지명]이야.
5. 폴더명3 이 [아티팩트명]이야

수집된 정보와 인터넷에서 다운받은 파일로 로컬 파일 내용을 수정해야해.
6. "https://paper.pe.kr/sts-template/mvc/pom.xml"을 다운받아. 그리고 로컬 현재 작업 폴더의 pom.xml을 덮어쓰기해.
7. 덮어쓰기한 pom.xml의 내용을 수정해야 해.
	- $${project}를 찾아서 [프로젝트명]으로 수정해
	- $${group}를 찾아서 [패키지명]으로 수정해
	- $${artifact}를 찾아서 [아티팩트명]으로 수정해
8. "https://paper.pe.kr/sts-template/mvc/web.xml"을 다운받아. 그리고 "로컬 현재 작업 폴더/src/main/webapp/WEB-INF/web.xml"을 덮어쓰기 해.
9. "https://paper.pe.kr/sts-template/mvc/root-context.xml"을 다운받아. 그리고 "로컬 현재 작업 폴더/src/main/webapp/WEB-INF/spring/root-context.xml"을 덮어쓰기 해.
10. "https://paper.pe.kr/sts-template/mvc/servlet-context.xml"을 다운받아. 그리고 "로컬 현재 작업 폴더/src/main/webapp/WEB-INF/spring/appServlet/servlet-context.xml"을 덮어쓰기 해.
11. "https://paper.pe.kr/sts-template/mvc/logback.xml"을 다운받아. 그리고 "로컬 현재 작업 폴더/src/main/resources/logback.xml"로 이동 해.
12. "https://paper.pe.kr/sts-template/mvc/log4jdbc.log4j2.properties"을 다운받아. 그리고 "로컬 현재 작업 폴더/src/main/resources/log4jdbc.log4j2.properties"로 이동 해.
13. "로컬 현재 작업 폴더/src/main/resources/log4j.xml"을 삭제해

이런 작업을 수행하고, 단계별 로그를 찍는 bat를 만들어줘


14. "로컬 현재 작업 폴더/src/main/java/폴더명1/폴더명2/폴더명/HomeController.java" 파일을 삭제해.
15. "로컬 현재 작업 폴더/src/main/webapp/WEB-INF/views/index.jsp" 파일을 삭제해.
16. "로컬 현재 작업 폴더/src/main/webapp/WEB-INF/spring/appServlet/servlet-context.xml"에서 $${group}를 찾아서 [패키지명]으로 수정해
17. "로컬 현재 작업 폴더/src/main/webapp/WEB-INF/spring/appServlet/servlet-context.xml"에서 $${artifact}를 찾아서 [아티팩트명]으로 수정해
